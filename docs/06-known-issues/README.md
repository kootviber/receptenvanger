# 06 - Known Issues

Gebruik deze map voor structurele of terugkerende problemen.

## Leg per issue vast
- Symptoom
- Oorzaak
- Oplossing
- Preventie
- Tags

## Wanneer toevoegen?
- Incident dat langer dan een dag relevant blijft.
- CI/CD of dependency-probleem dat vaker terugkomt.
- Hosting of browserbeperking met duidelijke impact.

Gebruik `templates/known-issue-template.md` als startpunt.

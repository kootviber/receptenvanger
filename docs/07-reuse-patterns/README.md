# 07 - Reuse Patterns

Gebruik deze map voor bewezen patronen die in meerdere projecten bruikbaar zijn.

## Per patroon vastleggen
- Probleem en context
- Oplossingsaanpak
- Wanneer wel gebruiken
- Wanneer niet gebruiken
- Valkuilen
- Link(s) naar voorbeeldcode

## Beginlijst patronen
- Auth flow
- Protected route
- CRUD table
- Dashboard shell
- Form validation pattern
- Upload pattern
- Webhook verification pattern
- Retry/backoff pattern

Gebruik `templates/reuse-pattern-template.md` als startpunt.

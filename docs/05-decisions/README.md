# 05 - Decisions (ADR)

Gebruik deze map voor Architecture Decision Records (ADR's).

## Wanneer hoort een keuze hier thuis?
- Wijzigingen met impact op architectuur, security, data of deployment.
- Keuzes die moeilijk terug te draaien zijn.
- Nieuwe standaard die meerdere features raakt.

## Minimale inhoud
- Context
- Beslissing
- Alternatieven
- Gevolgen

Gebruik `templates/adr-template.md` als startpunt.
